# gameKit 

gameKit is my template to quickly make third and first person narrative games, especially for game jams. 

Assets used:
- Maaack's Game Template
- Terrain3D
- FuncGodot
- Jeheno Third-Person Controller
- Sprouty Dialogs

Goals:
- Compatible with Windows, macOS, Linux, iOS, Android, and Web exports (for portfolio purposes)
- Small library of 3D assets and textures pre-loaded
- Very easy level and environment building, preferablly entirely in-engine
- Spatial sound design and music system, preferablly entirely in-engine
- Dialog system that supports branching options, either text or voice acting, inspired by Kentucky Route Zero or Firewatch, preferablly entirely in-engine
